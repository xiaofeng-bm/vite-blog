# hooks 简介

hooks 的诞生是为了解决之前函数组件没有生命周期，无法存在内部状态的问题。

## hooks 带来的好处

- 1、逻辑复用，重复的逻辑代码可以拆成一个单独的 hooks
- 2、有助于关注分离，简单的说就是把相关的代码可以放在一块，类比 vue 中的 Composition API
